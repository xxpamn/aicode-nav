# 项目决策记录 - AI Code Education Navigator

## 项目定位
- **域名**: aicode-nav.com
- **定位**: AI编程教育工具垂直导航站
- **目标用户**: 想学习编程/提升编程技能的人（尤其是初学者和转行者）
- **核心价值**: 帮助用户找到最适合的AI辅助编程学习工具

## 为什么选这个方向
1. 编程教育市场持续增长，AI工具降低了学习门槛
2. 大量新手不知道用什么工具学习
3. 竞争度低于通用AI导航站
4. 教育类工具佣金率高（30-50%常见）

## 网站结构
- 首页：工具分类展示 + 热门推荐
- 分类页：按场景分类（入门学习、代码练习、项目实战、面试准备）
- 评测页：深度工具评测 + 使用教程
- 博客页：学习资源、行业动态

## 技术栈
- 框架: Next.js 14 (App Router)
- 样式: Tailwind CSS
- 部署: Vercel (免费)
- 数据库: 初期用JSON文件，后期可上Supabase
