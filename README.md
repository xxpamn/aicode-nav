## 🚀 快速开始

### 本地预览

```bash
# 进入项目目录
cd aicode-nav

# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve

# 访问 http://localhost:8000
```

## 🤝 贡献指南

欢迎提交新的AI编程工具信息、测评文章或使用教程。

### 提交新工具
1. 在 `js/data.js` 中添加工具数据
2. 创建工具详情页 `tools/[tool-id].html`
3. 提交Pull Request

### 提交测评
1. 在 `posts/` 目录创建新的HTML文件
2. 遵循现有文章格式
3. 提交Pull Request
