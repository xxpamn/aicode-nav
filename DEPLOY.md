# AICode Nav 部署指南

## 域名信息
- **域名**: aicode-nav.com
- **项目路径**: `/workspace/aicode-nav`

## 部署方案推荐

### 方案一：Vercel（推荐）
**优势**: 免费、自动部署、全球CDN、支持自定义域名

**步骤**:
1. 注册 Vercel 账号 (vercel.com)
2. 将项目代码推送到 GitHub
3. 在 Vercel 中导入 GitHub 仓库
4. 配置自定义域名 aicode-nav.com
5. 在域名注册商处添加 DNS 解析记录

**DNS配置**:
```
类型: CNAME
主机: @
记录值: cname.vercel-dns.com
```

### 方案二：Cloudflare Pages
**优势**: 免费、全球CDN、内置SEO优化

**步骤**:
1. 注册 Cloudflare 账号
2. 进入 Pages 服务
3. 连接 GitHub 仓库或直接上传
4. 配置自定义域名
5. 域名DNS切换到Cloudflare

### 方案三：GitHub Pages
**优势**: 完全免费、与GitHub集成

**步骤**:
1. 创建 GitHub 仓库
2. 推送代码到仓库
3. 开启 GitHub Pages 功能
4. 配置自定义域名

**注意**: GitHub Pages 对国内访问速度一般

### 方案四：国内服务器/对象存储
**优势**: 国内访问速度快、SEO友好

**推荐平台**:
- 阿里云 OSS + CDN
- 腾讯云 COS + CDN
- 又拍云

**步骤**:
1. 购买对象存储服务
2. 上传网站文件
3. 开启静态网站托管
4. 配置CDN加速
5. 绑定域名

## 域名解析配置

无论选择哪种方案，都需要在域名注册商处配置DNS：

1. 登录你的域名注册商后台
2. 找到DNS解析/域名管理
3. 添加以下记录：

**如果使用 Vercel/Cloudflare**:
```
类型: CNAME
主机: @ 或 www
记录值: [平台提供的地址]
TTL: 600
```

**如果使用国内服务器**:
```
类型: A
主机: @
记录值: [服务器IP]
TTL: 600
```

## 推荐配置

**最佳选择**: Vercel + Cloudflare DNS
- 免费且稳定
- 全球访问速度快
- HTTPS自动配置
- 自动部署

## SSL证书

所有推荐方案都自动提供SSL证书，无需额外配置。
