### 2. 部署网站
有两个选择：

**方案A：Vercel（推荐，最简单）**
1. 访问 https://github.com/new 创建新仓库
2. 上传项目代码
3. 访问 https://vercel.com 用GitHub登录
4. 导入项目，一键部署
5. 绑定域名 aicode-nav.com

**方案B：Cloudflare Pages（国内访问快）**
1. 访问 https://dash.cloudflare.com
2. 进入 Pages → 创建项目
3. 连接GitHub仓库
4. 构建命令: `npm run build`
5. 输出目录: `dist`
6. 绑定域名
